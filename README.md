# Code and Data for Networked Parrondo Games

This repository contains the code, cached numerical arrays, CSV outputs, and
generated figure files associated with the manuscript:

**Analytical Theory of Parrondo's Paradox on Complex Networks: Necessary
Condition, Critical Penalty, and Topological Robustness**

Repository URL:
<https://github.com/wanghuilin8/parrondo-network-theory-code-data>

No external experimental or observational datasets are used. The numerical
data in this repository are generated by the simulation and figure-generation
scripts listed below.

## Contents

- `generate_figures.py`: regenerates the publication-style figures from the
  complete one-node analytical calculations, cached stationary Monte Carlo
  arrays, and simulation outputs.
- `run_robustness_simulations.py`: runs the additional robustness simulations
  used for Fig. 7 and writes CSV/text summaries to `data/`.
- `requirements.txt`: Python dependencies.
- `data/figure_data_cache.npz`: cached numerical arrays used by the
  figure-generation script.
- `data/additional_robustness_curves.csv`: CSV output for the additional
  robustness checks.
- `data/additional_robustness_summary.txt`: text summary of the additional
  robustness windows.
- `figures/`: generated figure files used in the revised manuscript.

## Reproducing the Results

Use Python 3.10 or newer. From the repository root, run:

```bash
python -m pip install -r requirements.txt
python run_robustness_simulations.py
python generate_figures.py
```

The simulation scripts fix the random seeds. Monte Carlo fitness is accumulated
only after a burn-in phase, and stationary trap occupancy is sampled after
burn-in. The figure-generation script writes the complete figure set to
`figures/`.

## Notes

The requirements file pins NumPy below version 2 to avoid binary compatibility
issues with older Matplotlib wheels. The manuscript LaTeX source, response
letter, and cover letter are not included in this repository archive.
