# Reproducibility Notes

This folder contains the simulation scripts, figure-generation scripts,
cached numerical arrays, CSV outputs, and generated figure files needed to
reproduce the simulations and figures reported in the article.

## Files

- `run_robustness_simulations.py`: deterministic Monte Carlo script for Fig. 7.
- `generate_figures.py`: publication-style figure script used to redraw
  Figs. 1--7 with consistent typography, line weights, legends, and
  color palette.
- `data/figure_data_cache.npz`: cached stationary Monte Carlo data for
  the complete one-node theory validation, WS robustness, and BA
  scale-free figures.
- `data/additional_robustness_curves.csv`: raw simulation output
  generated by `run_robustness_simulations.py`.
- `data/additional_robustness_summary.txt`: window summary for
  the additional checks.
- `requirements.txt`: Python packages required by the scripts.

## Run

```bash
python -m pip install -r requirements.txt
python run_robustness_simulations.py
python generate_figures.py
```

The requirements file pins NumPy below version 2 to avoid binary
compatibility issues with older Matplotlib wheels.

The scripts use fixed random seeds. Monte Carlo fitness is accumulated only
after a burn-in phase, and stationary trap occupancy is sampled after burn-in.
`run_robustness_simulations.py` writes the additional robustness CSV and
summary files in `data/`. `generate_figures.py` loads cached Monte Carlo data
when available and regenerates the full figure set in `figures/`.
